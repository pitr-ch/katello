TYPES =

The types hash below was introduced to simplify rendering of the status of various tasks/actions. It was initially introduced to support System Event history and was later refactored, so that it may be used for system groups...etc. Refer to the job.rb and system_task.rb for examples of it's usage.

{
    #package tasks
   :package_install => {
        :english_name =>N_("Package Install"),
        :type => :package,
        # in the event_messages structure, the first element is used for status when the action/event is scheduled
        # (e.g. System->Packages pane, the second and third are used on the Event History for singular/plural
        :event_messages => {
            :running => [N_('Installing Package...'), N_('installing package...'), N_('installing packages...')],
